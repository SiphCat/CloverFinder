Clovermedia

The live community gallery runs in the Next.js app. Start the dev server and open /clovermedia (for example http://127.0.0.1:3000/clovermedia).

Explorers opt in when posting a find with a photo. Run supabase/sql/cloverfinder_finds_badges_media.sql in Supabase so uploads work.
